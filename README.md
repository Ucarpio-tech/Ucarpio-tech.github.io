# Ucarpio-tech.github.io

Landing page de **VIDRALIX**, empresa de cristales y aluminios en República Dominicana.

Sitio estático publicado con GitHub Pages en [vidralixrd.com](https://www.vidralixrd.com).

## Estructura

- `index.html` — marcado y contenido de la página.
- `style.css` — estilos.
- `script.js` — carrusel de proyectos destacados.
- `assets/` — imágenes, logo e íconos favicon.
- `CNAME` — dominio personalizado para GitHub Pages.

## Desarrollo local

No requiere build ni dependencias. Basta con abrir `index.html` en el navegador o servir la carpeta con cualquier servidor estático, por ejemplo:

```bash
python3 -m http.server 8000
```
